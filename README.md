# 🎓 Student Database SQL Project

This project demonstrates basic SQL operations using a Student Database.

## 📊 Features Covered
- Database creation
- Table creation
- Data insertion
- Data retrieval (SELECT)
- Table modification (ALTER)
- Updating records
- Deleting records
- Dropping tables
- Creating new tables

## 🗂️ Tables Used

### Students Table
- StudentID (Primary Key)
- Name
- Age
- Department
- CGPA

### Courses Table
- CourseID (Primary Key)
- CourseName
- Credits

## 🛠️ Tools Used
- SQL Server (SSMS)

## 🚀 Learning Outcome
This project helped in understanding fundamental SQL queries and database management operations.
